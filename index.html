Частые вопросы

Спишут ли деньги сразу? +
Нет, первый платеж (449 ₽) будет только через 45 дней. Для проверки карты Яндекс может заморозить 11 рублей, которые сразу же вернутся на счет.
Как отключить, чтобы не списали деньги потом? +
Зайди в личный кабинет Яндекс ID -> Подписки -> Управление. Там можно нажать «Отменить подписку». Подписка продолжит работать до конца бесплатных 45 дней.
Работает ли код для старых аккаунтов? +
Да, промокод работает как для новых пользователей, так и для тех, у кого сейчас нет активной подписки Плюс.
